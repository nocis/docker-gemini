# Gemini Docker Sandbox Template

This repository provides a Docker-based sandbox template for a localized and isolated Google Gemini CLI experience, minimizing host system risks. It is designed to bring you directly into a container pre-configured for the Gemini CLI.

## Getting Started

To set up and enter the Gemini CLI sandbox, simply run the provided script:

```bash
./run_gemini_cli.sh
```

This script handles the Docker setup, builds the container, and places you directly into the Gemini CLI environment.

## Features (Managed by the script)

*   Isolated Gemini CLI environment
*   Persistent configuration
